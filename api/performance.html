실시간 성과 조회

광고 계정 실시간 성과 조회

특정 광고 계정의 실시간 성과를 조회합니다. 오늘 또는 어제 데이터를 조회할 수 있으며, 조회 결과는 업데이트 시점을 기준으로 오름차순 정렬되어 제공됩니다.

응답의 until 필드 값은 마지막으로 조회된 타임스탬프를 나타냅니다. 해당 값을 다음 요청의 since 파라미터로 포함시키면 이전 요청 이후에 업데이트된 소재만을 조회할 수 있습니다.

path Parameters
version
required
float
Default: "1.0"

API 버전

adAccountNo
required
integer <int64>
Example: 1234

광고 계정 번호

query Parameters
targetDate
string
Enum: "today" "yesterday"

조회 대상 일자 ( 오늘 : today, 어제 : yesterday ), 주어지지 않으면 today

since
integer <int64>
Example: since=1675907423

특정 시점 이후 성과가 업데이트된 소재만을 조회하기 위한 타임스탬프

header Parameters
AccessManagerAccountNo
any

해당 광고 계정의 성과에 대한 접근 권한이 있는 관리 계정 번호

Responses

http://openapi.naver.com/v1/ad-api/{version}/adAccounts/{adAccountNo}/performance/realtime/creatives

과거 성과 조회

광고 계정 과거 성과 조회

특정 광고 계정의 과거 성과를 조회합니다. 캠페인, 광고 그룹, 소재 단위로 집계할 수 있으며, 조회 결과는 날짜, 광고주, 캠페인, 광고 그룹, 소재 번호 순으로 오름차순 정렬되어 제공됩니다.

조회 기간은 최대 31일이며, 2년 전부터 전일 데이터까지 조회가 가능합니다. 전일 데이터는 금일 02시부터 조회가 가능합니다.

한 번에 최대 1000개 까지 조회가 가능하며, 이를 초과하는 개수가 존재하는 경우 페이징이 적용됩니다.

응답의 next 필드 값을 다음 요청의 next 파라미터로 포함시키면 다음 페이지를 조회할 수 있으며, 해당 토큰은 10분간 유효합니다.

version
required
float
Default: "1.0"

API 버전

required
integer <int64>
Example: 1234

광고 계정 번호

aggregationType
required
string
Enum: "campaigns" "adSets" "creatives"

성과 제공 단위 ( 캠페인 : campaigns, 광고 그룹 : adSets, 소재 : creatives )

query Parameters
startDate
required
string
Example: startDate=2023-01-01

조회 시작일 (yyyy-MM-dd 형식)

endDate
required
string
Example: endDate=2023-01-30

조회 종료일 (yyyy-MM-dd 형식)

next
string

다음 페이지 조회를 위한 토큰

limit
integer <int32>
Example: limit=1000

조회할 레코드 수, 주어지지 않으면 1000

header Parameters
AccessManagerAccountNo
any

해당 광고 계정의 성과에 대한 접근 권한이 있는 관리 계정 번호

Responses
