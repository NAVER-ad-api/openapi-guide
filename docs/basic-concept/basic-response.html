Page​

  • API 스펙 문서 응답 값 중 Page 관련 공통 설명 입니다.

값타입내용

pageInteger페이지 번호

totalPagesInteger전체 페이지 수

lastboolean마지막 페이지 여부

응답 코드​

HTTP 상태 코드​

코드의미설명
200OK성공적으로 수행되었습니다.
400Bad Request잘못된 요청을 보낸 경우 발생합니다.
401Unauthorized인증이 실패했습니다.
403Forbidden필요한 권한이 없어 수행할 수 없습니다.
422Unprocessable Entity요청 포멧은 올바르나, 성공 응답을 내려줄 수 없는 경우 입니다.
429Too Many Requests사용자가 일정 시간 동안 너무 많은 요청을 보냈습니다.
500Internal Server Error서버 오류 입니다. 재요청해도 계속 발생하면, 문의가 필요합니다.

응답 케이스별 설명​

  • 400 Bad Request
    • 요청 파라미터를 수정해서 다시 호출해야 됩니다.
{
    "error": {
        "code": "BAD_REQUEST",
        "message": "Bad Request",
        "description": "광고계정(1017) 하위에 속하지 않은 광고그룹([123])이 존재합니다."
    }
}
  • 422 Unprocessable Entity
    • detailCode : 상황별로 코드 값이 정의되어 있습니다. 상세 코드 설명을 참고 하시면 되겠습니다.
{
    "error": {
        "code": "CONDITION_DOES_NOT_MEET",
        "detailCode": "AD__BID_PRICE_OUT_OF_RANGE",
        "message": "입찰가 설정 범위를 벗어났습니다.",
        "description": "허용하는 최소 금액: 10, 최대 금액: 1000000, 입력값(adSetNo: 99145, bidPrice: 10000000000000)"
    }
}

코드​

  • code 설명입니다.
  • HTTP 상태 코드에서 몇가지로 세분화 되었습니다.
HTTP 상태 코드code
400BAD_REQUEST
401AUTH_ERROR
422CONDITION_DOES_NOT_MEET
DATA_NOT_EXIST
INVALID_PARAMETER
500UNKNOWN_ERROR

상세 코드​

  • detailCode 설명입니다.
  • 해당 에러가 발생시 대처할 수 있는 방안에 대해서 설명합니다.
코드설명
AD__BID_STRATEGY_VALUE_OUT_OF_RANGE입찰 한도 설정 범위를 벗어났습니다.
AD__BID_PRICE_OUT_OF_RANGE입찰가 설정 범위를 벗어났습니다.
AD__AD_SET_BUDGET_OUT_OF_RANGE허용된 '광고 그룹 예산' 설정 범위를 벗어났습니다.
AD__AD_SET_CANNOT_CREATE_EXCEED_MAX생성 가능한 최대 개수에 도달하여 새 광고 그룹을 생성할 수 없습니다.
AD__TARGETING_CANNOT_SET_AGE타겟팅을 설정할 수 없는 연령입니다.
- 성인 타겟팅의 경우 미성년자 연령을 포함시킬 수 없습니다.
AD__BID_STRATEGY_COST_CAP_NEED_AGREE비용 한도 사용 시 동의가 필요합니다.
- 광고계정no 기준으로 1번만 동의하시면 됩니다.
- 동의하는 방법 : gfa.naver.com 에 접속하여, 광고그룹 생성시 해당 옵션을 선택하고 동의 체크를 해서 저장합니다.
AD__BID_STRATEGY_NO_CAP_NEED_AGREE입찰가 한도 없음 사용 시 동의가 필요합니다.
- 광고계정no 기준으로 1번만 동의하시면 됩니다.
- 동의하는 방법 : gfa.naver.com 에 접속하여, 광고그룹 생성시 해당 옵션을 선택하고 동의 체크를 해서 저장합니다.
AD__START_TIME_CANNOT_CHANGE_IN_OPERATION광고가 운영중이거나 종료된 경우 시작일시를 변경할 수 없습니다.
AD__AD_SET_CANNOT_SET_SOME_TARGETINGS해당 매체에 허용하지 않는 타겟팅을 설정하셨습니다.
STATS__INVALID_API_VERSIONAPI 버전이 올바르지 않습니다.
