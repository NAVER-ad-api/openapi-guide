네이버 로그인 애플리케이션

네이버 로그인 - OAuth​

네이버 로그인은 OAuth 2.0 기반의 사용자 인증 기능을 제공해 네이버가 아닌 다른 서비스에서 네이버의 사용자 인증 기능을 이용할 수 있게 하는 서비스입니다. 네이버 로그인 개발 가이드

애플리케이션 등록​

네이버 오픈API를 사용하려면 먼저 네이버 개발자 센터에서 애플리케이션을 등록하고 클라이언트 아이디와 클라이언트 시크릿을 발급받아야 합니다.

  1. 네이버 개발자 센터의 메뉴에서 Application > 애플리케이션 등록을 선택합니다.
  2. 이용약관 동의 단계에서 이용약관에 동의합니다.를 선택한 다음 확인을 클릭합니다.
  3. 계정 정보 등록 단계에서 휴대폰 인증을 완료하고 회사 이름을 입력한 다음 확인을 클릭합니다. 휴대폰 인증은 담당자 연락처 확인을 위해 필요한 과정이며, 애플리케이션을 처음 등록할 때 한 번만 인증받으면 됩니다.

API 사용 > '네이버 로그인' 선택​

  • 사용 API 중에서 '네이버 로그인' 을 반드시 선택합니다.
  1. 애플리케이션 등록 (API이용신청) 페이지에서 애플리케이션 등록 세부 정보를 입력한 다음 등록하기를 클릭합니다.

네이버 성과형 디스플레이 광고 API 사용 신청​

등록된 내 애플리케이션에서 Client ID 값을 복사하고, 네이버 성과형 디스플레이 광고 사이트에 접속하여 대표 관리 계정 > 설정 > API 관리 메뉴의 "API 사용 신청" 을 통해 Client ID 를 등록해 주시면, 관리자 확인 후 API 사용 권한이 부여될 수 있습니다. (단, 테스트 기간 동안은 일부 관리 계정에서만 API 사용 신청 이 가능합니다.)

애플리케이션 개발​

튜토리얼 > Web 애플리케이션 을 참고하여 애플리케이션을 개발합니다. 네이버 아이디로 로그인하여 액세스토큰을 발급받을 수 있습니다.
