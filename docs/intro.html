시작하기

네이버 성과형 디스플레이 API 는 베타 테스트 중으로, 공식 파트너사인 경우에 한해 API 사용 신청 및 권한 부여가 가능합니다. 공식 파트너사가 아닌 경우 API 사용 신청이 제한되므로 참고 부탁드리겠습니다.

개요​

네이버 성과형 디스플레이 광고 API는 성과형 디스플레이 광고를 프로그래밍 방식으로 관리하고 조회할 수 있는 HTTP REST 인터페이스 입니다.

시작 전 준비​

시작하기 전에 네이버 개발자 센터에서 애플리케이션을 등록하고 로그인 애플리케이션을 개발하는 과정이 필요합니다.

  • 애플리케이션 등록
  • 애플리케이션 개발

용어 및 인터페이스​

성과형 광고에서 사용하는 용어들은 API에서 아래 인터페이스로 사용할 수 있습니다.
